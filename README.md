# 가까운 병원 찾기 🏥

보훈복지공단 위탁병원 현황을 기반으로 한 병원 찾기 웹 애플리케이션

## 주요 기능

1. **병원 지도 표시**: 전국 보훈병원 및 위탁병원을 지도에 마커로 표시
2. **병원 상세 정보**: 병상 수, 진료과, 병원 종류, 주소 등 상세 정보 제공
3. **위치 기반 검색**: 사용자의 현재 위치에서 가까운 병원 찾기
4. **거리 계산**: 내 위치에서 각 병원까지의 거리 표시
5. **병원 종류별 필터**: 종합병원, 병원, 의원으로 필터링 가능
6. **교통편 찾기**: 카카오맵 연동으로 길찾기 기능 제공

- 서울중앙보훈병원
- 부산보훈병원
- 대구보훈병원
- 광주보훈병원
- 대전보훈병원
- 인천보훈병원
- 청주위탁병원
- 천안위탁병원
- 울산위탁병원
- 경기남부위탁병원
- 강릉위탁의원
- 제주위탁의원

## 사용 방법

1. 웹 브라우저로 `index.html` 파일 열기
2. "📍 내 위치에서 병원 찾기" 버튼 클릭하여 위치 권한 허용
3. 병원 목록에서 원하는 병원 클릭하여 상세 정보 확인
4. "🚗 교통편 찾기" 버튼으로 카카오맵에서 길찾기

## 기술 스택

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla)
- Canvas API (지도 렌더링)
- Geolocation API (위치 정보)
